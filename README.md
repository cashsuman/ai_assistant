ROJECT NAME
============
AI Assistant Chatbot using Groq API

PROJECT DESCRIPTION
===================
The AI Assistant Chatbot is a Python-based conversational application
that uses the Groq API and Large Language Models (LLMs) to provide
intelligent responses to user queries.

FEATURES
========
• Interactive command-line chatbot
• Groq API integration
• Fast AI-generated responses
• Secure API key management
• Error handling
• Customizable AI model

TECH STACK
==========
Programming Language:
- Python 3.x

Libraries:
- groq
- python-dotenv
- os
- datetime

AI Model:
- Llama 3.3 70B Versatile

SETUP INSTRUCTIONS
==================
1. Clone the Repository
   git clone <repository-url>

2. Create Virtual Environment
   python -m venv .env

3. Activate Environment
   Windows:
   .env\\Scripts\\activate

4. Install Packages
   pip install groq python-dotenv

ENVIRONMENT VARIABLES
=====================
Create a .env file:

GROQ_API_KEY=your_groq_api_key_here
MODEL_NAME=llama-3.3-70b-versatile

HOW TO RUN
==========
python chatbot.py

EXAMPLE INPUT
=============
User: What is Machine Learning?

EXAMPLE OUTPUT

that enables computers to learn from data and improve
